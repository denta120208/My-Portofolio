About Me

Saya adalah siswa Metland School jurusan PPLG (Pengembangan Perangkat Lunak dan Gim) yang memiliki minat besar di bidang pemrograman, khususnya sebagai Front-End dan Back-End Developer, dengan fokus utama pada Web Development. Saya belajar coding secara otodidak melalui berbagai sumber di internet dan terus mengembangkan keterampilan saya di bidang ini. Beberapa teknologi dan bahasa pemrograman yang telah saya pelajari meliputi: HTML, CSS, dan JavaScript untuk pengembangan tampilan antarmuka (front-end) PHP dan Laravel untuk pengembangan sisi server (back-end) Python (termasuk library seperti Pandas), C, dan Kotlin Arduino untuk proyek berbasis IoT Construct dan After Effects untuk pengembangan game dan animasi Saya aktif membuat berbagai proyek pribadi untuk terus mengasah kemampuan saya dan memperdalam pemahaman terhadap teknologi yang saya pelajari. Saya juga cukup berpengalaman dalam menggunakan Laravel sebagai framework utama untuk membangun aplikasi web yang efisien dan terstruktur.
